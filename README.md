## 👋 About '모호재'

저는 변명보다는 해결책을 찾는 것을 중요하게 생각하는 게임 클라이언트 개발자입니다.  
저는 제 약점을 파악하고 그것을 강점으로 바꾸는 것을 즐기며, 다른 사람들이 함께 일하고 싶어하는 개발자로 성장하는 것을 목표로 합니다.
